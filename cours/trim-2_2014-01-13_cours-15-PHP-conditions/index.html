Conditions

IF

IF

If permet de tester qu'une condition soit vraie

Fonctionne comme en javascript


<?php
    $foo = true;
    if($foo == true)
    {
        echo 'Vraie !';
    }

    // 'Vraie !' va s'afficher
?>

IF

Lorsqu'il n'y a qu'une seule instruction, il n'est pas nécessaire d'écrire les accolades { }


<?php
    $foo = true;
    if($foo == true)
        echo 'Vraie !';

    // 'Vraie !' va s'afficher
?>

IF

Si l'on test que la valeur doit être true ou false il n'est pas nécessaire de l'écrire


<?php
    if($foo == true)
        echo 'Vraie !';

    // Équivaut à 
    if($foo)
        echo 'Vraie !';
?>
                    

<?php
    if($foo == false)
        echo 'Faux !';

    // Équivaut à 
    if(!$foo)
        echo 'Faux !';
?>

IF

La triple égalité === permet de tester la valeur ET le type


<?php
    $foo = 1;
    if($foo === true)
        echo 'Vraie !';

    // 'Vraie !' ne va pas s'afficher car true n'est pas du même type que 1
?>

IF

Exercice


<?php
    if('1' == 1)
        echo 'Vraie !';

    if('1' === 1)
        echo 'Vraie !';
        
    if('1' !== 1)
        echo 'Vraie !';
        
    if('1' != 1)
        echo 'Vraie !';
?>

IF ternaire

Il est possible de se passer du IF


<?php
    $foo = true;
    echo $foo ? 'oui' : 'non';
?>

ELSE

Comme en JS, ELSE et déclenché quand le IF ne l'est pas


<?php
    $foo = false;
    if($foo)
    {
        echo 'dans le IF';
    }
    else
    {
        echo 'dans le ELSE';
    }
?>

SWITCH ... CASE

Switch ... Case

Le format suivant peut-être écrit plus simplement avec un switch


<?php
    $foo = 3;
    if($foo == 1)
    {
        echo 'On fait ça';
    }
    else if($foo == 2)
    {
        echo 'Ou alors ceci';
    }
    else if($foo == 3)
    {
        echo 'Ou encore cela';
    }
    else
    {
        echo 'Sinon ça';
    }
?>

Switch ... Case

Avec le switch


<?php
    $foo = 3;
    switch($foo)
    {
        case 1:
            echo 'On fait ça';
            break;
        case 2:
            echo 'Ou alors ceci';
            break;
        case 3:
            echo 'Ou encore cela';
            break;
        default:
            echo 'Sinon ça';
    }
?>
