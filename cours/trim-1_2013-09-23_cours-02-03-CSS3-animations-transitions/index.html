Animations et transitions

Les animations de ces slides sont faites avec via des transitions CSS3

Awesome...

Différence entre transition et animation

La transition se fait entre un état A et un état B

L'animation se fait le long d'une timeline

Transitions

Transitions

Comment l'utiliser

  • transition-property (obligatoire)
    Propriétés CSS qui vont subir les transitions
  • transition-duration (obligatoire)
    Durée de la transition (s/ms)
  • transition-timing-function
    Type de transition (ease,linear,ease-in,ease-out,ease-in-out)
  • transition-delay
    Retard avant le départ (s/ms)
Transitions

Exemple CSS3


.transition{
    transition-property        : left;
    transition-duration        : 1s;
    transition-timing-function : ease;
    transition-delay           : 0s;
}
					

.transition{
    transition:left 1s ease 0s;
}

Transitions

Exemple CSS3


.transition{
	transition-property        : all;
	transition-duration        : 2s;
	transition-timing-function : ease-out;
	transition-delay           : 1s;
}
					

.transition{
    transition:all 2s ease-out 1s;
}

Animations

C'est comme les transitions

Mais avec un contrôle dans le temps

Animations

Comment l'utiliser

  • animation-name (obligatoire)
    Nom de l'animation, on voit ça plus tard
  • animation-duration (obligatoire, déjà vu)
    Durée de l'animation
  • animation-timing-function (déjà vu)
    Type de transition (ease, linear, ease-in, ease-out, ease-in-out, personnalisée)
  • animation-delay (déjà vu)
    Retard avant le départ (s/ms)
Animations

Comment l'utiliser

  • animation-iteration-count
    Combien de fois jouer l'animation (1 par défaut)
  • animation-direction
    Direction de l'animation (normal, reverse, alternate, alternate-reverse)
  • animation-play-state
    État par défaut de l'animation (running, paused)
Animations

Exemple CSS3


.animation{
	animation-name     : monanimation;
	animation-duration : 3s;
}
					

.animation{
    animation : monanimation 3s;
}
					

@keyframes monanimation {
    0%   {left:0px;}
    50%  {left:800px;}
    70%  {left:250px;}
    100% {left:500px;}
}
